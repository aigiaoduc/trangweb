Lưu ý quan trọng

Để đảm bảo quá trình cài đặt và kích hoạt diễn ra suôn sẻ, bạn nên tắt tạm thời các chương trình bảo mật như phần mềm diệt virus và Windows Defender.

Đây là hướng dẫn sử dụng bản vá (crack) để kích hoạt phần mềm. Vui lòng cân nhắc trước khi sử dụng.

Các bước cài đặt chi tiết

1

Chạy file Setup.exe

Chạy file Setup.exe để bắt đầu cài đặt. Làm theo các bước hướng dẫn trên màn hình cho đến khi quá trình cài đặt hoàn tất. Không khởi động phần mềm sau khi cài xong.

2

Sao chép file Patch

Mở thư mục Cr@ck, sao chép file Patch.exe vào thư mục cài đặt phần mềm. Thư mục cài đặt thường nằm tại:

  • C:\Program Files\[Tên phần mềm]
  • C:\Program Files (x86)\[Tên phần mềm]
3

Chạy Patch với quyền Admin

Nhấp chuột phải vào file Patch.exe, chọn “Run as Administrator”.

4

Kích hoạt

Nhấn vào nút “Patch” (hình đầu con báo). Chờ đến khi hiện thông báo hoàn tất quá trình.

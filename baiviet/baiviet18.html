2 Cách tắt Windows Security (Windows Defender) trên Windows 11

Phần mềm diệt virus miễn phí và mạnh mẽ từ Microsoft, nhưng đôi khi bạn cần tạm thời vô hiệu hóa nó.

Windows Security là một phần mềm diệt virus miễn phí và được cài đặt sẵn trên các dòng máy tính chạy hệ điều hành của nhà Windows và bao gồm cả Windows 11. Với giao diện và cách thao tác hoàn toàn mới như vậy, nếu bạn chưa biết cách tắt phần mềm diệt virus thì bài viết này sẽ hướng dẫn cho bạn cách tắt Windows Security (Windows Defender) trên Windows 11.

I. Tắt trong Cài đặt (Settings)

1. Hướng dẫn nhanh

  • Mở Cài đặt (Settings)
  • Chọn Privacy & Security → Windows Security
  • Chọn Virus & threat protection
  • Chọn Manage Settings tại mục Virus & threat protection settings
  • Tắt công tắc mục Real-time Protection để tắt Windows Security

2. Hướng dẫn chi tiết

Bước 1: Vào Settings > chọn Privacy & Security > chọn tiếp mục Windows Security.

Bước 2: Tiếp theo, bạn chọn Virus & threat protection.

Bước 3: Sau đó, tại mục Virus & threat protection settings > bạn chọn Manage Settings.

Bước 4: Cuối cùng, tắt công tắc mục Real-time Protection để tắt Windows Security.

II. Dùng Defender Control

1. Hướng dẫn nhanh

  • Tải phần mềm Defender Control 2.0 tại đây.
  • Giải nén file vừa tải.
  • Mở tệp dfControl.exe > Nhập Password là "sordum" > nhấn OK.
  • Chọn 1 trong 2 nút: Disable Windows Defender (Tắt) hoặc Enable Windows Defender (Mở).

2. Hướng dẫn chi tiết

Bước 1: Tải phần mềm Defender Control 2.0 tại đây.

Bước 2: Tại file vừa tải về, bạn click chuột phải và chọn Extract All... để giải nén file.

Bước 3: Mở file vừa giải nén > Mở tệp dfControl.exe > Nhập Password là "sordum" > nhấn OK.

Bước 4: Sau khi mở phần mềm, bạn chọn 1 trong 2 nút: Disable Windows Defender (Tắt) hoặc Enable Windows Defender (Mở).
